### commit标记说明：

| 标记            | 说明                                  |
| --------------- | ------------------------------------- |
| feature/feat    | 新功能开发                            |
| bug/fix/bug-fix | bug修复                               |
| refactor/perf   | 重构代码/优化配置&参数/优化逻辑和功能 |
| test            | 添加单元测试用例相关                  |
| docs            | 添加文档                              |
| info            | 添加注释类信息                        |
| format          | 不修改业务逻辑，仅做代码规范格式化    |
| merge           | 仅做分支合并同步                      |
| depend          | 构建脚本,任务等相关代码               |
| chore           | 构建脚本任务等相关代码                |
| del             | 删除等破坏性操作                      |
